# Discovery Agent

Repositorio base para organizar procesos de discovery.

## Estructura

- `.claude/scripts/`: scripts auxiliares para automatizar tareas.
- `discoveries/_template/`: plantilla para nuevos discoveries.
- `discoveries/<nombre>/interviews/`: entrevistas, notas crudas y evidencia.
- `discoveries/<nombre>/outputs/`: entregables, resúmenes y resultados finales.

## Reflexion breve

### ¿Que supuesto tuyo sobre el problema cambio al revisar la evidencia real?

Pense que el problema era solo digitalizar la agenda. La evidencia mostro que lo importante era que todos confiaran en una sola fuente real.

### ¿Que fue lo mas dificil de cumplir la regla de "cero invencion"?

Lo mas dificil fue no inventar respuestas cuando faltaba informacion.

### ¿Para que te serviria la idea de "gobernanza ejecutable" en tu trabajo?

Me serviria para tomar decisiones con reglas claras y comprobables.
